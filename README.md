🏨 HotelVerdap - Sistema de Gerenciamento
📌 Sobre o Projeto

O HotelVerdap é um sistema desenvolvido em Kotlin que simula a administração de um hotel, permitindo gerenciar clientes, reservas, eventos e serviços adicionais.

O projeto foi criado com o objetivo de praticar conceitos de programação orientada a objetos (POO), organização em camadas e lógica de negócios em um cenário real.

⚙️ Funcionalidades

O sistema oferece as seguintes funcionalidades:

✅ Cadastro de usuários no sistema

🔐 Login de usuários

👤 Registro de clientes

🛏️ Registro de quartos escolhidos pelo usuário

🎉 Inscrição de usuários em eventos do hotel

❄️ Solicitação de ar-condicionado com empresas parceiras

⛽ Registro de abastecimento de veículos do hotel

🛠️ Tecnologias e Ferramentas
